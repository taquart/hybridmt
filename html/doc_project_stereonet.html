project_stereonet

Project data onto the stereonet plot.

Contents

  • Syntax
  • Description
  • Parameters
  • See also

Syntax

[X,Y] = project_stereonet(AZIMUTH, TAKEOFF)
[X,Y] = project_stereonet(AZIMUTH, TAKEOFF, 'PropertyName',PropertyValue, ...)
[X, Y, R] = project_stereonet(...)

Description

Use project_stereonet to project input AZIMUTH and TAKEOFF data onto the stereonet. The value of AZIMUTH is measured from North towards East. The value of TAKEOFF angle is measured from BOTTOM towards TOP. By default, the equal area projection is used to project the points on stereonet.

focimt(Input) reads input data and performs moment tensor inversion using fociMT software and default processing parameters. The input array can be either a path to the formatted ASCII input file or Input cell array.

[X,Y] = project_stereonet(AZM, TKO) takes azimuths and takeoff angles and returns the X and Y matrices of the same size as AZIMUTH and TAKEOFF with coordinates projected using the default Schmidt projection.

[X,Y] = project_stereonet(AZIMUTH, TAKEOFF, 'PropertyName',PropertyValue, ...) allows to specify optional parameters.

[X, Y, R] = project_stereonet(...) returns also the radii of projected points.

Parameters

Projection - Stereographic projection

'schmidt' (default) | 'wullf'

Defines projection type for graphical representation of the seismic moment tensor inversion (beach ball). The user may use 'schmidt' (equal area) or 'wullf' (equal angle) projection to plot the beach ball.

Scale - Plotting scale

1.0 (default) | positive scalar value

Defines the maximum radius of created plot.
