readvel1d

Read input ASCII file in fociMT 1D velocity model format

Contents

  • Syntax
  • Description
  • Examples
  • See also

Syntax

Input = readvel1d(filename)

Description

Use readvel1d(filename) to read ASCII file in fociMT 1D velocity format containing input data for the seismic moment tensor inversion. The resulting Input cell array contains input data organized in a cell array. The cell array may be used as input parameter for the following seismic moment tensor inversion with focimt.m or seismic moment tensor refinement procedure hybridmt.m.

The full description of ASCII input file in 1D velocity model focimt format is available here.

Examples

This example reads input data in fociMT 1D velocity format and stores them in Input cell array:

Input = readvel1d('./examples/example2_1d.txt');
